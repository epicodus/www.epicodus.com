About the course

This course provides the foundations for building websites using the tools and technologies behind sites like WhiteHouse.gov and Sony.com. You'll learn everything from building powerful content management backends to creating interactive frontend user interfaces.

At Epicodus, our approach is to start by developing strong programming fundamentals using a minimal number of tools and libraries. The first half of the classwork will get you familiar with the basics of PHP, JavaScript, the web, and databases.

After you have a solid foundation for reasoning about your code, we'll add libraries and frameworks that are used in the real world to make you more productive. In the second half of the classwork, you'll use tools like AngularJS, Ember.js, and Drupal.

At the end of the course, we'll match you with local companies for two-month paid internships.

Be sure to also read the more general information on our courses.

What you'll learn

Here's some of what we'll cover:

  • HTML and CSS: the presentation and styling languages of the web
  • PHP: the most popular language for web servers
  • SQL: the database language to store and retrieve information
  • jQuery: a library for interactive web pages
  • JavaScript: the programming language of web browsers
  • AngularJS: a framework for building interactive web pages
  • Drupal: a content management system powering millions of websites

Don't worry if you don't understand what all of this means; that's what Epicodus is for.

Before starting

You don't need any prior education or experience for Epicodus. That said, we expect you to try out programming basics on your own before applying, to make sure that you actually like it. Our litmus test to make sure you've done enough background work is that we ask you to solve the Ping-Pong test and the layout test before or shortly after applying. A good place to start learning is the Epicodus curriculum at www.learnhowtoprogram.com; it will guide you through everything you need to know to solve the tests.
