About the course

This course provides the foundations for building web applications using the tools and technologies behind websites like Airbnb and Square. You'll learn everything from building powerful backend server applications to creating interactive frontend user interfaces.

At Epicodus, our approach is to start by developing strong programming fundamentals using a minimal number of tools and libraries. The first half of the classwork will get you familiar with the basics of Ruby, JavaScript, the web, and databases.

After you have a solid foundation for reasoning about your code, we'll add libraries and frameworks that are used in the real world to make you more productive. In the second half of the classwork, you'll use tools like AngularJS, Ember.js, and Ruby on Rails.

For the last month of the course, you'll intern with a local company.

Be sure to also read the more general information on our courses.

What you'll learn

Here's some of what we'll cover:

  • HTML and CSS: the presentation and styling languages of the web
  • Ruby: an elegant language for the server
  • SQL: the database language to store and retrieve information
  • Active Record: a Ruby library to make databases easy
  • jQuery: a library for interactive web pages
  • JavaScript: the programming language of web browsers
  • AngularJS and Ember.js: frameworks for building interactive web pages
  • Ruby on Rails: a framework to make web applications easy
  • AJAX: a way to make web pages interactive
  • APIs: the way that web applications talk to each other

Don't worry if you don't understand what all of this means; that's what Epicodus is for.

Before starting

You don't need any prior education or experience for Epicodus. That said, we expect you to try out programming basics on your own before applying, to make sure that you actually like it. Our litmus test to make sure you've done enough background work is that we ask you to solve the Ping-Pong test and the layout test before or shortly after applying. A good place to start learning is the Epicodus curriculum at www.learnhowtoprogram.com; it will guide you through everything you need to know to solve the tests.
